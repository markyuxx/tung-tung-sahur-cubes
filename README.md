# Tung Tung Sahur Cubes

Tung Tung Sahur Cubes is a browser puzzle game inspired by falling-block games and the Tung Tung Tung Sahur meme.

The goal is to fill the glowing Tung Tung Sahur silhouette with golden cubes before the 10-minute timer runs out. Extra cubes are allowed, but the game ends when the solver detects that no future combination of pieces can complete the target shape.

## Features

- Falling golden cube pieces
- Tung Tung Sahur silhouette objective
- 10-minute run timer
- Automatic impossible-state detection
- Win and lose preview buttons
- Animated Tung Tung Sahur meme card
- Background music and keyboard toggle
- Responsive desktop and mobile controls

## Controls

| Action | Keys |
| --- | --- |
| Move | Arrow Left / Arrow Right / A / D |
| Rotate | Arrow Up / W / Space |
| Hard drop | Enter |
| Pause | P |
| Music | M or the Sound button |

## Run Locally

Open `index.html` in a browser, or serve the folder with any static web server.

## Deploy

This project is static and can be deployed directly to Vercel.
